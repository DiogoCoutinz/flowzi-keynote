# Flowzi Keynote

Apresentacao interativa "Do Zero aos +4.000€" — Marcelo & Diogo, Flowzi.

## Stack

- Vite + React + TypeScript
- Tailwind CSS + shadcn/ui

## Desenvolvimento

```sh
npm install
npm run dev
```

## Controlos da apresentacao

- **Setas** / **Scroll** / **Swipe** — Navegar slides
- **F** — Fullscreen
- **Esc** — Sair de fullscreen
